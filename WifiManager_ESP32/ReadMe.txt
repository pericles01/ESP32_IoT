The IoT device can connect every where.
How it works:
1- the device scan and search for knowed wifi connexions to connect
2- when none was found, the device create an Access Point and allows the User to connect to it.
3- the user give its wifi identifiants to the device to connect on its wifi
4- every time the device is on, it follows the same steps.
